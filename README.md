# Moa Clay Collection

## Development

From your terminal:

```sh
yarn  dev
```

## Stage

From your terminal:

```sh
yarn build
./node_modules/vercel/dist/index.js
./node_modules/vercel/dist/index.js alias `build` stage.moaclayco.com
```

## Productio

From your terminal:

```sh
yarn build
./node_modules/vercel/dist/index.js --prod
```
